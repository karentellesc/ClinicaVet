## Sistema de gerenciamento de uma clinica veterinária

Esse projeto foi desenvolvido em C# com o objetivo de simular um sistema de uma clinica veterinária, onde o usuário cadastra/atualiza/remove clientes (pets) e consultas do sistema. 
Os dados cadastrados no sistema são armazenados de forma automática no banco de dados MySQL.

# Funcionalidades do sistema
- Cadastrar clientes (dados do animal e do dono)
- Atualizar informações do cliente
- Cadastrar consultas e procedimentos realizados
- Apagar clientes
- Listar clientes e as consultas realizadas
